# 👨‍💻 About Me

I am a Cybersecurity student with a focused interest in penetration testing and offensive security. My goal is to build a professional career as a Penetration Tester by continuously developing practical skills in web application security, network security, and vulnerability assessment.

Alongside my academic studies, I actively improve my hands-on experience through labs and self-study. In my free time, I participate in bug bounty programs to strengthen my ability to identify, analyze, and responsibly report security vulnerabilities.
